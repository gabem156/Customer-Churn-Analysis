# Customer Churn Analysis

## Project Overview
This project analyzes customer churn data to identify the key factors influencing customer retention and customer loss. Using Python for data cleaning, exploratory data analysis (EDA), and visualization, the project uncovers patterns that can help businesses improve customer satisfaction and reduce churn rates.

The analysis focuses on customer demographics, contract types, service usage, monthly charges, and tenure behavior.

---

## Business Problem
Customer churn directly impacts company revenue and long-term growth. Businesses need to understand why customers leave and identify high-risk customer groups in order to improve retention strategies and customer experience.

This project aims to:
- Identify churn patterns
- Analyze customer behavior
- Discover high-risk customer segments
- Generate actionable business recommendations

## Dataset
The dataset contains customer information for a telecommunications company, including:
- Customer demographics
- Internet services
- Contract types
- Billing information
- Monthly charges
- Customer tenure
- Churn status

---

## Data Cleaning
The following cleaning steps were performed:
- Converted data types for numerical analysis
- Handled missing values
- Removed invalid records
- Verified dataset consistency
- Prepared data for visualization and analysis

---

## Exploratory Data Analysis (EDA)
The analysis explored:
- Churn distribution
- Customer demographics
- Contract type behavior
- Monthly charge trends
- Internet service patterns
- Customer tenure analysis
- Correlation analysis

----

## Key Insights
- Customers with month-to-month contracts showed significantly higher churn rates.
- Customers with higher monthly charges were more likely to leave the service.
- New customers with shorter tenure had elevated churn behavior.
- Fiber optic internet users demonstrated higher churn levels compared to other service types.
- Long-term contract customers showed stronger retention patterns.

---

## Business Recommendations
- Encourage customers to transition to long-term contracts through incentives.
- Improve onboarding experiences for new customers to reduce early churn.
- Investigate pricing concerns among customers with high monthly charges.
- Develop targeted retention campaigns for high-risk customer segments.
- Enhance customer support strategies for customers with shorter tenure.

## Conclusion

This project demonstrates how data analytics can be used to identify customer churn drivers and support data-driven business decisions. Through data cleaning, visualization, and exploratory analysis, the project provides actionable insights that can help improve customer retention strategies.
