## Data

The dataset used in this project contains customer information from a telecommunications company and is focused on analyzing customer churn behavior.

### Dataset Features
The dataset includes:
- Customer demographics
- Contract types
- Internet services
- Payment methods
- Monthly charges
- Total charges
- Customer tenure
- Churn status

### Data Files

#### Raw Dataset
Contains the original customer churn data before preprocessing and cleaning.

#### Cleaned Dataset
Contains the processed dataset after:
- Handling missing values
- Converting data types
- Removing inconsistencies
- Preparing data for analysis and visualization

The cleaned dataset was used for exploratory data analysis (EDA) and dashboard development.
